## Expression DNA (Data-Driven)

Based on statistical analysis of 4,449 tweets:

- **38% of tweets are under 50 characters** — minimalism is the default
- **Emoji as punctuation**: 😂 (495x), 🙏 (309x), 🤷‍♂️ (188x)
- **"4" appears 374 times** — his identity signature
- **Post-prison behavioral shift**: Reply rate 23% → 64%, emoji usage 9% → 20%, Bitcoin mentions 10.6% → 4.6%, Education mentions 1.9% → 6.4%
